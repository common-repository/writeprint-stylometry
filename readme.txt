=== Plugin Name ===
Writeprint Stylometry
Author: Rodrigo S. Alhadeff
Tags: writeprint, comments, admin, authors
Requires at least: 2.8.0
Tested up to: 3.0.1
Stable tag: 1.3

== Description ==

Writeprint Stylometry plugin lets you analyze comments in order to compare and identify authorship

== Installation ==

1. Upload `writeprint-stylometry` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The utility can be accessed under the "Comments" Menu of the WordPress Admin.
